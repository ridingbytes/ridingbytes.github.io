UIUX

 

UI/UX

These two terms stand for User Interface and User Experience. While the first one is mainly concerned with the layout of application elements, the latter one cares about how the interaction with the application feels to the user.

Both terms are linked to each other and highly influence the user acceptance of the finished application/product.

Case Study: Bika Spotlight Search

Bika Spotlight Search offers an MAC OS X like Search UI for Bika LIMS. The overlay uses theS modern JavaScript Framework BackboneJS.

Case Study: Plone Commander

Plone Commander offers an alternative UI for the CMS Plone. The application follows a modern mobile first approach using ExtJS.
