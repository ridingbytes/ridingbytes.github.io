SENAITE Open Source LIMS Project

  • suitable for all types of laboratories
  • flexible architecture
  • you design the system together with us

SENAITE is an Open Source LIMS for enterprise environments.
It is especially focused on high speed, excellent performance and stability. It currently ships with an integrated JSON API, has a user friendly GUI and provides rich UI components. It is built on top of Plone CMS with Python as its main programming language.

Plone Open Source CMS Project

Plone is among the top 2% of all open source projects worldwide and supported by more than 350 solution providers in more than 100 countries. The project has been actively developed since 2001, is available in more than 40 languages, and has the best security track record of any major CMS.
