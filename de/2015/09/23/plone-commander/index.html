Plone Commander

Plone Commander is a cross platform application built with ExtJS and Electron. It allows you to find, view, edit and explore Plone contents with ease.

It is free, open source and MIT licensed.

What is it about?

Plone Commander is basically a proof of concept, if the Plone API is capable of handling an external application like this.

Where to get?

Plone Commander is available on the official RIDING BYTES GitHub Account.
