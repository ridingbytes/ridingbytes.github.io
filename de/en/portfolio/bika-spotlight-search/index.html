Bika Spotlight Search

Project Data
  • Project Category: BIKA LIMS BACKBONEJS OPEN-SOURCE
  • Client: Open Source
  • Start Date: 2017-02-20
  • End Date: ongoing
Project Features
  • Intuitive Search & Navigation Overlay
  • Supports Bika LIMS >= 3.2
  • MAC OS X like Spotlight Search
  • Lightning Fast

Overview

Bika Spotlight Search is a modern JavaScript application written with CoffeeScript and based on the BackboneJS framework. It communicates with Bika LIMS through plone.jsonapi.routes.
