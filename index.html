Plone JSON API

JSON API for the Plone CMS

plone.jsonapi is a web interface to the Plone CMS. It allows developers to expose their content information over HTTP in the JSON format. It facilitates the integration of external web services with Plone.

  • Client: Plone Community
  • Category: API Development

Plone Explorer

Plone Explorer Application

Plone Explorer is a web based application to interact with the Plone CMS over the JSON API. It is build on the powerful ExtJS Framework and combines the flexibility of a web application with the power of a native program.

  • Client: Plone Community
  • Category: Web Application Development
